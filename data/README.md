# Data

Le dataset utilisé est disponible publiquement sur Kaggle :

🔗 [Airline Passengers Dataset](https://www.kaggle.com/datasets/vedatgul/airline-passengers)


## Description

Ce jeu de données contient des données temporelles liées au transport aérien.

* **Type de données :** Série temporelle (*Time series*)
* **Portée :** Nombre mensuel de passagers aériens sur une période spécifique.
* **Structure :** Série temporelle univariée avec une fréquence régulière et continue (mensuelle).

### Structure des colonnes

| Colonne | Type de données | Description |
| :--- | :--- | :--- |
| `Month` | Date | Information temporelle (fréquence mensuelle). |
| `Passengers` | Numérique | Nombre de passagers. |
